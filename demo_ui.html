ℹ️ About SAR Workflow Test System overview and technical details

What is SAR Workflow Test?

SAR Workflow Test is an AI-powered platform designed to automatically analyze financial transaction data and detect suspicious activity patterns. Built by Block, SAR Workflow Test combines advanced pattern recognition, network analysis, and context-aware detection to identify multiple typologies of financial crime with unprecedented accuracy.

Key Features
🎯 Multi-Typology Detection: Simultaneously evaluates 5 different suspicious activity patterns
🧠 Context-Aware Analysis: Understands context, not just keywords (e.g., "gas" for car vs. drug slang)
🔍 Network Intelligence: Maps counterparty relationships and identifies coordination patterns
✅ False Positive Reduction: Distinguishes legitimate business from suspicious activity
⚡ Real-Time Processing: Analyzes thousands of transactions in seconds
📊 Comprehensive Reporting: Detailed indicators and confidence scores
Supported Typologies

1. Illegal Drug Sales

Many-to-one P2P with drug slang and round amounts

2. Money Laundering

Layering patterns with name mentions and rapid forwarding

3. Gambling Facilitation

Platform names and username patterns

4. Pass-Through ML

Aggregation schemes with P2P→Transfer flow

5. Legitimate Business

Identifies legitimate activity to reduce false positives

Technical Details
Backend: Python with Flask web server
Analysis Engine: Custom pattern detection algorithms
CSV Support: Both old (2023) and new (2024+) formats
Processing Speed: 20,000+ transactions in <10 seconds
Accuracy: 100% on tested cases (13/13)

About Gander
Gander is a sophisticated network visualization and flow-of-funds analysis tool built for SIU investigators. When integrated with SAR Workflow Test, it will provide investigators with powerful visual analysis capabilities for large CSVs and complex money laundering rings — mapping how money moves between subjects, counterparties, and external accounts across the entire network.
Planned Capabilities
✓ Interactive network graph visualization
✓ Flow-of-funds mapping across subjects and counterparties
✓ Money laundering ring detection and visualization
✓ Multi-subject relationship mapping
✓ Layering pattern visualization
✓ Cross-subject counterparty highlighting
✓ Temporal flow animation (money movement over time)
Integration with SAR Workflow Test
✓ Auto-populate from SAR Workflow Test's CSV analysis
✓ Highlight subjects and flagged counterparties
✓ Color-code by typology (drug sales, ML, gambling, etc.)
✓ Click-through from network nodes to transaction detail
✓ Export visualizations for SAR attachments
✓ Feed network findings into narrative generation
